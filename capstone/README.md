# Capstone

This folder contains my main **capstone incident case study**, written in a SOC-style format and based on lab/simulated environments.

## Current capstone

- Capstone #1: `YYYY-MM-DD_capstone_<incident-type>_<short-title>.md`

## Capstone structure

My capstone uses the following sections:

- Executive summary
- Environment & telemetry
- Incident timeline
- Investigation narrative
  - Initial detection & triage
  - Scoping
  - Root cause
  - Containment / eradication / recovery
- Evidence pack
- Queries used
- Post-incident improvements (detection / prevention / process)
- Reflection
